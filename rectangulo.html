¿De dónde sale el área del triángulo?

Una exploración visual: construimos un rectángulo, lo cortamos con una diagonal y descubrimos por qué el área del triángulo es la mitad de base por altura.

Partimos de un rectángulo

Observamos un rectángulo de base b y altura h. Su área se calcula multiplicando base por altura.

Área del rectángulo b · h = 40 u²
Área de cada triángulo b · h / 2 = 20 u²
Conclusión A = (b · h) / 2
